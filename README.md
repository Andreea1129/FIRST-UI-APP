# 📌 Project Overview
This project is a static clone of the IT School website, built entirely using HTML5 and CSS3, without any frameworks or JavaScript.

The goal of this project was to practice semantic HTML structure, responsive design, layout techniques (Flexbox), and clean UI implementation.

The website includes multiple interconnected pages and simulates a real educational platform structure.

# 🚀 Features
🏠 Homepage:
- Navigation to all other pages
- Course listing preview
- 4 testimonial cards
- Partners website links

📚 Courses Page:
- All courses displayed as responsive cards
- Details for each course
- Button navigation to dedicated course pages

🎓 Individual Course Pages:
- Course descriptions with images
- Course overview section (certifications, price, trainer, etc)
- Technology used section (each technology links to its own dedicated page)

👥 About Us Page:
- Description section with images
- Trainer cards
- "Why our trainers" section

📞 Contact Page:
- Contact form layout
- Contact information section

# 🛠 Technologies Used
- HTML5
- CSS3

# 🎨 CSS Features & Concepts Used
- Flexbox for layout
- Responsive design with Media Queries
- CSS variables

# 📱 Responsive Design
The website is responsive. Media queries are used to adjust:
- Layout structure (row → column)
- Image scaling
- Card grid responsiveness
- Typography sizing
